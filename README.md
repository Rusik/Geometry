Orthogonal range searching with kd-tree
=======================================

####File format####
	n
	(x, y)
	(x, y)
	...
	(x, y)
	m
	(x1, y1, x2, y2)
	(x1, y1, x2, y2)
	...
	(x1, y1, x2, y2)
  
####Points in rectangle####
	y2       -----------  
			|           |  
			|           |  
	y1       -----------  
			x1          x2  

####Run wit GUI####
	> java -jar 2dtree.jar -gui
